# Contribuyendo al AI Chatbot

¡Gracias por tu interés en contribuir a este proyecto! Este documento establece las pautas para contribuir al código y la documentación.

## Cómo contribuir

1.  **Haz un Fork** del repositorio.
2.  **Crea una rama** para tu funcionalidad o corrección de errores (`git checkout -b feature/nueva-funcionalidad`).
3.  **Realiza tus cambios** y asegúrate de seguir el estilo de código existente.
4.  **Haz Commit** de tus cambios (`git commit -m 'Añadir nueva funcionalidad'`).
5.  **Haz Push** a la rama (`git push origin feature/nueva-funcionalidad`).
6.  **Abre un Pull Request**.

## Reportar Problemas

Si encuentras un error o tienes una sugerencia, por favor abre un "Issue" en GitHub describiendo el problema detalladamente.

## Estándares de Código

- Seguimos las guías de estilo PEP 8 para Python.
- Mantén el código limpio y comentado donde sea necesario.
- Asegúrate de que las nuevas funcionalidades no rompan las existentes.

¡Esperamos tus contribuciones!
